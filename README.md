## How to use
1. Download 「**NitroCat**-main.zip」 and Unzip that.
2. put out 「**NitroCat**」folder and put in setting/extension file(comming soon)
3. Click to open **NitroCat**.html(in **NitroCat** folder)
4. Let's programming in editor!
